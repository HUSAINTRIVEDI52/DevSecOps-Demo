================================================================================
                         TROUBLESHOOTING
================================================================================

SonarQube Issues
================

Won't start:
  1. Check Docker: docker ps
  2. View logs: docker logs sonarqube-vulnerable-app
  3. Check ports: netstat -an | grep 9000
  4. Restart: docker-compose restart

Scanner fails:
  1. Ensure SonarQube is running
  2. Check token is valid
  3. Verify sonar-scanner installed
  4. Check sonar-project.properties

Quality gate fails:
  1. Review issues in dashboard
  2. Fix critical issues first
  3. Re-run scanner

Dependency Check Issues
=======================

Docker not found:
  1. Install Docker Desktop
  2. Verify: docker --version
  3. Start Docker daemon

Scan takes too long:
  1. First run downloads database (~500MB)
  2. Subsequent runs are faster
  3. Use --exclude to skip directories

Memory issues:
  1. Increase Docker memory
  2. Use --exclude to reduce scope
  3. Run on larger machine

GitHub Actions Issues
=====================

Workflow not triggering:
  1. Check branch name (main, master, develop)
  2. Verify workflow file syntax
  3. Check GitHub Actions enabled
  4. Review workflow logs

Scan fails in Actions:
  1. Check logs in Actions tab
  2. Verify Docker available
  3. Check resource limits
  4. Review error messages
