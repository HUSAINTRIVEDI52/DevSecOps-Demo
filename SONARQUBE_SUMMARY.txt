================================================================================
                         SETUP INSTRUCTIONS
================================================================================

STEP 1: Prerequisites
  ✓ Docker installed
  ✓ Docker Compose installed
  ✓ Node.js 14+ installed
  ✓ npm installed

STEP 2: Start SonarQube
  Option A: docker-compose up -d
  Option B: bash setup-sonarqube.sh (Linux/Mac)
  Option C: setup-sonarqube.bat (Windows)
  Option D: npm run sonar:setup

STEP 3: Wait for Startup
  • SonarQube takes 30-60 seconds to start
  • Check: curl http://localhost:9000/api/system/health

STEP 4: Access Dashboard
  • URL: http://localhost:9000
  • Username: admin
  • Password: admin

STEP 5: Generate Token
  1. Go to http://localhost:9000
  2. Click profile (top right)
  3. Select 'My Account' → 'Security'
  4. Click 'Generate Tokens'
  5. Enter name: 'vulnerable-app-token'
  6. Copy token

STEP 6: Run Scanner
  • npm run sonar
  • Or: sonar-scanner (if installed globally)

STEP 7: View Results
  • Go to http://localhost:9000/projects
  • Click 'vulnerable-app'
  • Review issues, metrics, quality gate
